## Tailles de texte Tailwind CSS

| Classe | Taille | Utilisation typique |
|--------|--------|---------------------|
| `text-xs` | 12px (0.75rem) | Mentions légales, labels très petits |
| `text-sm` | 14px (0.875rem) | Texte secondaire, descriptions ← **Votre choix actuel** |
| `text-base` | 16px (1rem) | Texte principal, paragraphes |
| `text-lg` | 18px (1.125rem) | Texte légèrement plus grand |
| `text-xl` | 20px (1.25rem) | Sous-titres |
| `text-2xl` | 24px (1.5rem) | Titres de section |
| `text-3xl` | 30px (1.875rem) | Titres importants |
| `text-4xl` | 36px (2.25rem) | Grands titres |
| `text-5xl` | 48px (3rem) | Titres très grands |
| `text-6xl` | 60px (3.75rem) | Héros, bannières |

---

## Exemples de combinaisons courantes
```tsx
<!-- Petit texte discret -->
<p className="text-xs text-gray-500">...</p>

<!-- Texte standard -->
<p className="text-sm text-gray-900">...</p>

<!-- Texte principal -->
<p className="text-base text-gray-800">...</p>

<!-- Sous-titre -->
<h3 className="text-lg font-semibold text-gray-900">...</h3>

<!-- Titre de section -->
<h2 className="text-xl font-bold text-gray-900">...</h2>
```

# Classes de transformation de texte (Text Transform)

| Classe | Effet | Exemple |
|--------|-------|---------|
| `uppercase` | TOUT EN MAJUSCULES | "BONJOUR" |
| `lowercase` | tout en minuscules | "bonjour" |
| `capitalize` | Première Lettre De Chaque Mot | "Bonjour Le Monde" |
| `normal-case` | Texte normal (annule les autres) | "Bonjour le monde" |

---

## Exemples d'utilisation

```tsx


Exemple :
{/* Tout en majuscules */}
<p className="text-sm text-gray-900 uppercase">
  Commencez par sélectionner une première annonce.
</p>

{/* Première lettre de chaque mot en majuscule */}
<p className="text-sm text-gray-900 capitalize">
  commencez par sélectionner une première annonce.
</p>

{/* Tout en minuscules */}
<p className="text-sm text-gray-900 lowercase">
  COMMENCEZ PAR SÉLECTIONNER UNE PREMIÈRE ANNONCE.
</p>

{/* Titre en majuscules avec espacement des lettres */}
<h2 className="text-lg font-bold uppercase tracking-wide">
  Éléments de la newsletter
</h2>

{/* Bouton avec texte capitalisé */}
<button className="text-sm font-medium capitalize">
  créer une nouvelle campagne
</button>



```

# Espacement des lettres (Letter Spacing)


| Classe | Effet |
|--------|-------|
| `tracking-tighter` | Lettres très serrées |
| `tracking-tight` | Lettres serrées |
| `tracking-normal` | Espacement normal |
| `tracking-wide` | Lettres espacées |
| `tracking-wider` | Lettres plus espacées |
| `tracking-widest` | Lettres très espacées |

# Classes font-weight Tailwind CSS

| Classe | CSS | Rendu |
|--------|-----|-------|
| `font-thin` | font-weight: 100 | Très fin |
| `font-extralight` | font-weight: 200 | Extra léger |
| `font-light` | font-weight: 300 | Léger |
| `font-normal` | font-weight: 400 | Normal (défaut) |
| `font-medium` | font-weight: 500 | Medium |
| `font-semibold` | font-weight: 600 | Semi-gras |
| `font-bold` | font-weight: 700 | Gras |
| `font-extrabold` | font-weight: 800 | Extra gras |
| `font-black` | font-weight: 900 | Très gras |
