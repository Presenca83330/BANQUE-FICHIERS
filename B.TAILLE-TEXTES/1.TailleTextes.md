## 2. Margin (Espacement exterieur)

### Prefixes

| Axe | Classe | Description |
|-----|--------|-------------|
| Tous | `m-{valeur}` | Margin sur les 4 cotes |
| Horizontal | `mx-{valeur}` | Margin gauche + droite |
| Vertical | `my-{valeur}` | Margin haut + bas |
| Haut | `mt-{valeur}` | Margin-top uniquement |
| Bas | `mb-{valeur}` | Margin-bottom uniquement |
| Gauche | `ml-{valeur}` | Margin-left uniquement |
| Droite | `mr-{valeur}` | Margin-right uniquement |

**Meme echelle que padding** + valeurs speciales :

| Classe | Effet |
|--------|-------|
| `m-auto` | Centrage automatique |
| `mx-auto` | Centrage horizontal |
| `my-auto` | Centrage vertical |
| `-mt-4` | Margin negative (-16px) |
| `-mb-8` | Margin negative bas (-32px) |

## 4. Gap (Espacement Flexbox/Grid)

| Classe | Description |
|--------|-------------|
| `gap-{valeur}` | Espacement entre tous les elements |
| `gap-x-{valeur}` | Espacement horizontal uniquement |
| `gap-y-{valeur}` | Espacement vertical uniquement |

**Meme echelle que padding/margin**

Exemples :
- `flex gap-4` → 16px entre chaque element
- `grid gap-6` → 24px entre cellules
- `flex gap-[60px]` → 60px (valeur arbitraire)

## 8. Exemples du projet Xoutos

```
CarteHero :
├── px-[120px] max-[900px]:px-4    → 120px desktop, 16px mobile
├── py-[100px] max-[900px]:py-10   → 100px desktop, 40px mobile
├── rounded-[24px] max-[900px]:rounded-[10px]
├── mb-6 max-[900px]:mb-4          → 24px desktop, 16px mobile
├── leading-[1.05]                 → ligne tres serree
└── tracking-[-3px]                → lettres serrees

Feature2Section :
├── py-20 max-[900px]:py-12        → 80px desktop, 48px mobile
├── px-[20px] max-[900px]:px-4     → 20px desktop, 16px mobile
├── gap-[60px] max-[900px]:gap-8   → 60px desktop, 32px mobile
├── pt-7                           → padding-top 28px
├── leading-[1.15]                 → hauteur ligne titres
└── rounded-[24px] max-[900px]:rounded-[16px]
```

## 9. Astuce : Pattern responsive

```jsx
// Desktop → Mobile
className="py-20 max-[900px]:py-12"     // 80px → 48px
className="px-[120px] max-[900px]:px-4" // 120px → 16px
className="gap-[60px] max-[900px]:gap-8" // 60px → 32px
className="text-[42px] max-[900px]:text-[26px]" // 42px → 26px
```
